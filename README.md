# sokoban-bevy
The famous Sokoban puzzle game with generated map, implemented using Bevy engine in Rust

Resources:

- Bevy guide: https://bevy-cheatbook.github.io/tutorial/guide.html
- online game: https://www.mathsisfun.com/games/sokoban.html
- sokoban skins: https://joriswit.nl/sokoban/en/skins.htm
- generating sokoban boards: https://github.com/mezpusz/sokohard
- implementation sokoban SDL in Rust: https://github.com/swatteau/sokoban-rs?tab=readme-ov-file
